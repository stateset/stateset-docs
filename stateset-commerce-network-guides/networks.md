# Networks

Stateset supports the following RPC protocols:

* URI over HTTP
* JSON-RPC 2.0 over HTTP

_**Stateset is currently in Testnet**_ (stateset-1-testnet)



Stateset Testnet Genesis File:

[https://rpc.stateset.zone/genesis](https://rpc.stateset.zone/genesis)



Official RPC Endpoints

`curl https://rpc.stateset.zone/abci_info?`

`curl https://rpc.stateset.zone/status?`



Official REST Endpoints

[https://rest-api.stateset.zone/](https://rest-api.stateset.zone/)
